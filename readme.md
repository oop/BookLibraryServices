## Description
A [Vivoo](https://vivoo.io) home assignment project.

## Installation
```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode 
$ npm run start:dev

# production
$ npm run start:prod
```

## Testing

```bash
#unit tests
$ npm test
```
